Novedades en Java 10

Desde Java 9 la poltica de Oracle ha sido lanzar una nueva versin de Java cada seis meses, de ah que las novedades en las ltimas versiones sean ms bien escasas respecto a la anterior.

En este documento vamos a describir algunas de las principales novedades que ha aparecido en la versin Java 10, en lo que a lenguaje y APIs de uso general se refiere.

Inferencia de tipos

A partir de Java 10, es posible declarar variables locales sin indicar el tipo de la misma, mediante el uso de la palabra reservada var:

var a=10;

A partir del valor asignado a la variable, el compilador "infiere" el tipo de la misma, en este caso int. La inferencia de tipos puede aplicarse a cualquier tipo Java, incluidas clases de cualquier tipo:

var datos=new ArrayList<Integer>();

Gracias a la inferencia de tipos, se evita tener que indicar nombres largos a la hora de definir el tipo de una variable cuando se trata de nombres de clases compuestos. Si se recomienda, a fin de claridad en el cdigo, que el nombre de la variable sea lo suficientemente claro como para deducir de qu tipo de dato se trata:

var connection=DriverManager.getConnection(....);

A la hora de aplicar la inferencia de tipos, hay que tener en cuenta las siguientes consideraciones:

  • Solo puede aplicarse con variables locales, nunca con atributos ni parmetros de mtodos.

  • La declaracin y asignacin de dato a la variable deben hacerse obligatoriamente en la misma instruccin, de manera que el compilador pueda determinar el tipo de la misma.

  • Al tratarse de una operacin en tiempo de compilacin, la inferencia de tipo no empeora ni mejora el rendimiento de la aplicacin en tiempo de ejecucin.

Mtodos de coleccin para creacin de copias

Las interfaces de coleccin List, Map y Set proporcionan un mtodo esttico copyOf() para crear copias no modificables de una coleccin. En el caso de List y Set, este mtodo recibe el objeto Collection que se quiere clonar:

List<E> copyOf(Collection<E> col) //mtodo de List

Set<E> copyOf(Collection<E> col) //mtodo de Set

Por su parte, en Map este mtodo recibe como parmetro la implementacin de Map que se quiere clonar:

Map<K,V> copyOf(Map<K,V> col) //mtodo de Map

En el siguiente ejemplo vemos como crear una lista a partir de otra:

ArrayList<Integer> enteros=new ArrayList<>();
enteros.add(20);
enteros.add(10);
enteros.add(35);
//creacin de copia no modificable
var copia=List.copyOf(enteros);
enteros.set(0, 4); //cambia la lista original, no la copia
//copia.set(0, 8); //esta operacin no podra realizarse, 
 //provocara una excepcin UnsupportedOperationException
for(Integer i:copia) {
 System.out.println(i);
}

Mtodos de reduccin de Collectors

La clase Collectors de Java 8 dispona de los mtodos toList(), toSet() y toMap() para reducir un un stream a coleccin. Ahora en Java 10 se han aadido toUnmodificableList(), toUnmodificableSet() y toUnmodificableMap() para generar colecciones no modificables.
